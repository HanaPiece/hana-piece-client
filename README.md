## 4. 개발 환경
- Version : Java 17
- IDE : IntelliJ
- Framework : SpringBoot 3.2.5
- Dependencies : JPA, QueryDSL, Spring security, jsonwebtoken, lombok, gson, jackson-databind-nullable

## 5. 주요 기능
- 목표 관리 - 목표 등록 및 목표 달성을 위한 상품 추천
- 계좌 관리 - 계좌 생성 및 관리
- 통장 쪼개기 - 사용 목적 별 통장 쪼개기 및 자동이체
- 상품 관리 - 상품 가입 및 자동이체
